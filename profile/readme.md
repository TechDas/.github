## What we do
- **Virtual Sensor Systems**
  - CO / 효율 / 품질 예측 모델
  - 실시간 상태 판정 및 알람
- **Monitoring & Diagnostics**
  - Sensor Validation (PCA, SPE, T²)
  - 공정 이상 탐지 및 원인 변수 식별
- **Industrial Data Pipeline**
  - RTDB / DB 수집
  - 전처리 → 모델 → 결과 저장/전송 → Dashboard
